# rsocket-cpp
C++ implementation of RSocket

 Coverage Status

Dependencies
Install folly:

brew install --HEAD folly
Building and running tests
After installing dependencies as above, you can build and run tests with:

# inside root ./rsocket-cpp
mkdir -p build
cd build
cmake -DCMAKE_BUILD_TYPE=DEBUG ../
make -j
./tests
